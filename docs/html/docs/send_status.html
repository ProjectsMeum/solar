function send_status¶

This function as attached as method to class class Car at initialization.

Warning

Correct configuration for emails must be set in solar.definitions.send_status_access.py. No email notification is sent if settings are incorrect of file is missing.

Function to send emails about the status or errors to user.

  • from_addr = ‘carnotification@mymail.com’

  • from_name = ‘Raspberry’

  • from_password = ‘notificationpassword’

  • to_addrs = ‘user@domain.com’

  • signature = ‘Mail from Raspberry’

# other values depending on mail server

  • port = 587

  • smtp_server = ‘smtp.gmail.com’

solar.send_status.send_status(msgstr: str, subject: str = None) → str[source]¶

send msgstr (with subject) to to_addrs
